# 美容室特化WEBテンプレート事業 要件定義書

---

## 1. 事業概要

### 1.1 目的
美容室・美容サロン向けに特化した高品質なWEBサイトテンプレートを制作し、個人経営〜10店舗程度のサロンに販売する。

### 1.2 ターゲット市場
- 日本の美容室：約25万店舗
- まともなHPを持っている：約3割（7.5万店舗）
- HPがないor古い：**約17.5万店舗** ← ここが市場

### 1.3 既存資産
- sample02（白基調・清潔感・モード系）← 設計書・画像完備
- Blacksmith Roastery派生（黒基調・無骨・スタイリッシュ）← カフェ版を美容室向けに調整

## 4. 技術スタック

### 4.1 使用技術

| 技術 | 用途 | 必須度 |
|------|------|:-----:|
| HTML5 | ページ構造 | 🔴 |
| CSS3（Flexbox / Grid） | デザイン・レイアウト | 🔴 |
| JavaScript（Vanilla JS） | タブ・ハンバーガー・アニメーション | 🔴 |
| レスポンシブデザイン | スマホ対応（メディアクエリ） | 🔴 |
| Google Maps iframe | 地図表示 | 🔴 |
| RESERVA埋め込み | 予約・問い合わせフォーム | 🟡 |
| GSAP（オプション） | リッチアニメーション | 🟢 |
| IntersectionObserver | スクロールリビール | 🟢 |

### 4.2 不要な技術

| 技術 | 理由 |
|------|------|
| React / Vue / Next.js | 静的なLPが基本。オーバーエンジニアリング |
| TypeScript | 美容室サイトには不要 |
| バックエンド（PHP / Node） | 予約・問い合わせは外部サービス利用 |
| データベース | 静的サイトで完結 |
| Astro | Blacksmithで使ったが、テンプレート量産には不向き |

## 5. 各テンプレートのファイル構成

```
beauty/templates/[template-name]/
├── index.html              # メインページ（1ページLP形式）
├── css/
│   └── style.css           # 全スタイル
├── js/
│   └── main.js             # 全スクリプト
├── images/
│   ├── hero.jpg            # ファーストビュー画像
│   ├── gallery-1.jpg       # 施術事例
│   ├── gallery-2.jpg
│   ├── gallery-3.jpg
│   ├── staff-1.jpg         # スタッフ写真
│   ├── staff-2.jpg
│   ├── staff-3.jpg
│   └── logo.png            # サロンロゴ（仮）
├── README.md               # テンプレート説明・カスタマイズ手順
└── preview.jpg             # テンプレートプレビュー画像
```

**1ファイル完結型の利点：**
- 納品が簡単（index.html1つ渡すだけ）
- サーバー管理不要（Amazon S3やNetlifyにポイ）
- 美容室オーナーが理解しやすい

## 8. 地図表示

### 8.1 実装方法

```html
<!-- Google Maps iframe埋め込み（3分で完了） -->
<div class="map-wrap" style="margin-top:24px; border-radius: 8px; overflow: hidden;">
  <iframe 
    src="https://www.google.com/maps/embed?pb=サロンID" 
    width="100%" 
    height="300" 
    style="border:0;" 
    allowfullscreen="" 
    loading="lazy">
  </iframe>
</div>
```

**ダークテーマ対応（Blacksmith方式）：**
```css
.map-wrap iframe {
  filter: grayscale(0.3) invert(0.92);
}
```

### 8.2 地図の取得手順（美容室オーナー向け手順書としても使える）

1. Googleマップで店名を検索
2. 左パネルの「共有」ボタンをクリック
3. 「地図を埋め込む」タブを選択
4. 表示されたiframeコードをコピー
5. テンプレートの`<div class="map-box">`内にペースト

## 8A. 問い合わせフォーム実装詳細

### 8A.1 方式別比較

| 方式 | 必要なもの | 難易度 | 料金 | 美容室への説明 |

| **Formspree** | Formspreeアカウント | 10分 | 無料〜月$5 | 英語画面なので美容室向け非推奨 |
| **自作PHP** | サーバー（PHP対応） | 30分 | サーバー代 | 保守が面倒 |

### 8A.2 RESERVA埋め込み（推奨）

美容室の大半はすでに **RESERVA（リザービア）** を導入済み。RESERVAには予約機能だけでなく、**お問い合わせフォーム機能も標準搭載**されている。

**実装手順：**
1. RESERVA管理画面にログイン
2. 「埋め込み」→「予約フォーム」のタグをコピー
3. テンプレートの`<section id="reserva">`内にペースト

```html
<!-- RESERVA埋め込み例（管理画面で発行されたコードに置き換え） -->
<iframe src="https://reserva.be/サロンID/reserve" width="100%" height="600" style="border:none;"></iframe>
```

### 8A.3 Googleフォーム埋め込み（RESERVA未導入の場合）

1. Googleフォーム（https://forms.google.com）にアクセス
2. 質問項目を作成（名前・メール・お問い合わせ内容）
3. 送信先メールアドレスを設定
4. 「送信」タブ → 「<>埋め込み」のHTMLをコピー
5. テンプレートの問い合わせセクションに埋め込み

```html
<!-- Googleフォーム埋め込み例 -->
<iframe src="https://docs.google.com/forms/d/e/フォームID/viewform?embedded=true" width="100%" height="600" style="border:none;"></iframe>
```
