# BLACK EDGE - 利用規約およびサポート規定

本パッケージ（Astro版 BLACK EDGE テンプレート）をクライアントへ販売、または受託案件として納品する際のライセンスおよびサポート範囲についての規定です。

## 1. ライセンスについて（買い切り・受託納品）
- 本テンプレートは、1回の購入（または受託契約）につき、1つのドメイン（1クライアント）でのみ使用を許可します。
- 納品したソースコードを別クライアントにそのまま流用して再販することは禁止とします（都度、新規案件としてのライセンス購入・契約が必要です）。

## 2. 料金に含まれる基本サポート範囲
「50万円クラス」の基本パッケージ料金には、以下の作業が含まれます。

- **初期セットアップ**: サーバーへのデプロイ（Vercel, Netlify等の静的ホスティング）、独自ドメイン設定。
- **テキスト・画像差し替え**: クライアントから提供された原稿（メニュー表、スタッフ情報等）および写真データの `site.json` への反映。
- **基本フォーム設定**: 指定されたメールアドレスへの Formspree（または同等サービス）経由での問い合わせフォーム連携。

## 3. 追加費用が発生するオプション
以下の要件については、基本料金外として別途お見積もりとなります。

- **プロカメラマンによる写真撮影**: 出張撮影、レタッチ作業（5〜10万円〜）。
- **大幅なデザイン変更**: レイアウトの大規模変更、アニメーション（GSAP/Lenis）のカスタム調整。
- **機能追加**: 予約システム（RESERVA以外の独自システム）のAPI連携、決済機能の追加。
- **納品後の保守運用**: 月額保守契約を結ばない場合の、納品1ヶ月後以降のテキスト修正・画像差し替え・レイアウト崩れの修正。

## 4. 納品後の無償保証期間
- 納品日から **1ヶ月間** を無償保証期間とします。
- 保証期間内における、明らかなバグ（表示崩れ、フォーム送信エラー等）や、軽微なテキスト修正（数カ所程度）は無償で対応いたします。
- クライアント都合による大規模なコンテンツ変更や仕様追加は、保証期間内であっても有償対応となります。
